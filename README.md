# EBIS Marketing

Маркетинговые материалы и исходники сайта для ООО «ЭБИС» — поставщик модульной металлочерепицы.

## Структура проекта

```
ads/              — Рекламные материалы (Яндекс.Директ): тексты, структура РК, изображения
landing/          — Одностраничный лендинг (Astro + Tailwind CSS)
```

## Стек (landing)

- [Astro](https://astro.build/)
- [Tailwind CSS](https://tailwindcss.com/)
- PHP (отправка форм)
